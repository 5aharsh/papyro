Papyro is a modern, lightweight, flat-file CMS built for speed and simplicity using FastAPI and Python.

## Philosophy
Content management should be direct and uncomplicated. Papyro eliminates the friction of bloated software and complex dependencies by providing a minimalist, flat-file architecture that ensures absolute control over data.

## Features
-   **Flat-file Storage**: No database required. Content is stored in standard Markdown files, ensuring portability and easy version control.
-   **FastAPI Powered**: High performance, async-first engine designed for modern web standards.
-   **Multi-Theming**: decouples presentation from content, allowing for instant theme switching.
-   **Dynamic Navigation**: Automatically generates site menus based on available static pages and configurable ordering.
-   **Live Reload**: Optimized for high-speed development with real-time feedback for content and theme changes.
-   **SEO Optimized**: Built with semantic HTML and configurable metadata to enhance search engine visibility.
